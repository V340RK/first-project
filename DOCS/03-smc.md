# Smart Money Concepts (SMC)

## Що таке SMC?

SMC — це набір концепцій про те, як **великі гравці** (банки, фонди, маркет-мейкери) рухають ринок.

Ідея проста: великі гравці не можуть просто "натиснути кнопку купити" — вони повинні **накопичити** позицію поступово, не злякавши ринок. Для цього вони використовують певні механізми, які залишають **сліди на графіку**.

SMC навчає читати ці сліди.

---

## Концепція 1: Market Structure (Структура ринку)

### Swing High і Swing Low
- **Swing High** — локальний максимум (ціна піднялась, потім впала)
- **Swing Low** — локальний мінімум (ціна впала, потім піднялась)

```
Приклад:
    /\        /\
   /  \      /  \
  /    \    /    \
 /      \  /      
         \/       
  SL    SH  SL    SH
```

### BOS — Break of Structure (Пробій структури)

Якщо ціна пробиває **попередній Swing High** — це BOS вгору: тренд продовжується.

```python
# BOS вгору: ціна закрилась вище попереднього swing high
bos_up = current_close > previous_swing_high

# BOS вниз: ціна закрилась нижче попереднього swing low
bos_down = current_close < previous_swing_low
```

### CHoCH — Change of Character (Зміна характеру ринку)

Якщо ринок в **аптренді** (зростання), але ціна раптом пробиває **попередній Swing Low** — це сигнал розвороту.

```
Аптренд:  вища вершина → вищий мінімум → вища вершина...
CHoCH:    раптово ціна пробиває попередній мінімум вниз
→ Сигнал: тренд змінюється на низхідний
```

**Для бота**: CHoCH на старшому таймфреймі (5хв, 15хв) визначає **напрямок торгівлі**.  
Тільки після бичачого CHoCH шукаємо лонги. Тільки після ведмежого — шорти.

## Концепція 2: Order Block (Блок ордерів)

### Що це?
Остання **протилежна** свічка перед сильним імпульсним рухом, який пробив структуру.

Простіше: велика риска гравця "зайшла" в ринок — і це залишило слід.

### Бичачий Order Block:
```
Знайти: остання ВЕДМЕЖА свічка перед сильним бичачим рухом вгору
Зона OB: від LOW до HIGH цієї ведмежої свічки

Коли ціна повертається в цю зону → чекаємо відскоку вгору
```

### Ведмежий Order Block:
```
Знайти: остання БИЧА свічка перед сильним ведмежим рухом вниз
Зона OB: від LOW до HIGH цієї бичачої свічки

Коли ціна повертається в цю зону → чекаємо відскоку вниз
```

### Алгоритм пошуку:
```python
def find_bullish_ob(candles):
    for i in range(len(candles) - 3, 0, -1):
        # Шукаємо ведмежу свічку (close < open)
        if candles[i].close < candles[i].open:
            # Перевіряємо чи після неї був сильний бичачий рух (BOS)
            if bos_up_detected_after(candles, i):
                return OBZone(low=candles[i].low, high=candles[i].high)
```

## Концепція 3: FVG — Fair Value Gap (Розрив справедливої ціни)

### Що це?
Коли ціна рухається так швидко, що утворюється **пропуск** між свічками — зона де не було нормальної торгівлі.

### Бичачий FVG:
```
Свічка [i-1]: максимум = 50,000
Свічка [i]:   великий бичачий стрибок
Свічка [i+1]: мінімум = 50,500

FVG зона: від 50,000 до 50,500
(між максимумом [i-1] і мінімумом [i+1])
```

Ціна повертається заповнити цей "пропуск" в ~70% випадків.

### Ведмежий FVG — дзеркально навпаки.

### Алгоритм:
```python
def find_fvg(candles):
    fvgs = []
    for i in range(1, len(candles) - 1):
        # Бичачий FVG
        if candles[i-1].high < candles[i+1].low:
            fvgs.append({
                'type': 'bullish',
                'top': candles[i+1].low,
                'bottom': candles[i-1].high
            })
        # Ведмежий FVG
        if candles[i-1].low > candles[i+1].high:
            fvgs.append({
                'type': 'bearish',
                'top': candles[i-1].low,
                'bottom': candles[i+1].high
            })
    return fvgs
```

## Концепція 4: Liquidity Sweep (Зачистка ліквідності)

### Що це?
Великі гравці знають де стоять стоп-лоси дрібних трейдерів (зазвичай трохи вище/нижче очевидних рівнів).  
Вони **спеціально** штовхають ціну туди, щоб забрати ліквідність, а потім розгортаються.

### Ознаки Liquidity Sweep:
```
1. Ціна тимчасово виходить за межі swing high/low (бере стопи)
2. Але ЗАКРИВАЄТЬСЯ назад в межах попереднього діапазону
3. Після цього сильний рух у протилежному напрямку

Приклад:
  Swing High = $51,000
  Ціна піднялась до $51,050 (взяла стопи вище $51,000)
  Але закрилась на $50,800
  → Liquidity Sweep → очікуємо падіння
```

## Як SMC поєднується з Order Flow сигналами

SMC дає нам **де** торгувати, Order Flow дає нам **коли** торгувати.

```
КРОК 1: SMC визначає контекст (старший таймфрейм, 15хв або 1год)
  → Визначаємо тренд через BOS/CHoCH
  → Знаходимо ключові зони: Order Block або FVG

КРОК 2: Чекаємо поки ціна прийде до зони (молодший таймфрейм, 1хв)

КРОК 3: Order Flow підтверджує реакцію в зоні
  → Stacked Imbalance на рівні OB/FVG
  → Absorption (хтось великий захищає рівень)
  → Delta Spike в напрямку тренду

КРОК 4: Вхід + Стоп + Тейк (дивись 04-risk-management.md)
```

### Приклад конкретного setup (Long):
```
15хв: Бичачий BOS → тренд вгору ✓
15хв: Знайшли бичачий Order Block на $49,500–$49,700 ✓
1хв:  Ціна прийшла до $49,600 (в зону OB) ✓
1хв:  Stacked Imbalance позитивний (3 рівні з перевагою покупців) ✓
1хв:  Delta Spike +500 (агресивні покупки) ✓

→ Всі умови виконані → Вхід LONG на $49,620
→ Стоп нижче зони OB: $49,450
→ Тейк: $50,050 (мінімум 1:2 R:R)
```
